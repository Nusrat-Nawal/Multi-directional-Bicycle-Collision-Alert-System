# Multi-directional-Bicycle-Collision-Alert-System
**Project Idea**
A multi-directional collision alert system designed for bicycle and rickshaw riders that detects vehicles approaching from the left, right, and rear sides and provides real-time warnings using LCD, buzzer, and a critical LED alert .

**Project Overview**
3 Ultrasonic Sensors (Left, Right, Back)
         ↓
Arduino processes data
         ↓ 
Measures distance and show collision risk in real-time through speed analysis
         ↓
Displays warnings on an LCD screen 
         ↓
Alerts are generated by buzzer when actual danger is detected
         ↓
Activates a red LED only in critical situations 

**Key Features**
1. Multi-Directional Detection System:
      i. 3 ultrasonic sensors (Left, Right, Back)
      ii. Covers all blind spots and provides full directional awareness

2. Real-Time Distance Analysis : 
      i. Measures distance in centimeters 
      ii. Detects closest vehicle and prioritizes highest risk direction

3. Speed Approximation:
      i. Calculates speed using distance change     
      ii. Fast decrease → High risk  and Slow decrease → Normal condition

4. Smart LCD Display (MAIN OUTPUT) :
      i. Displays all system information 
      ii. Shows multiple direction and distance clearly and provides real-time 
          safety messages

5. Buzzer Alert System:
      i. Safe = OFF
      ii. Warning = Slow beep
      iii. High Risk = Fast beep
      iv. Critical = Continuous loud sound

6. Critical Safety LED:
      i. Activates only in critical condition which 
        indicates emergency situation
      ii. Red LED ON + continuous buzzer 

7. False Alert Reduction System
      i. Uses multiple sensor readings and averages data for accuracy
      ii. Ignores sudden spikes

**Output States (LCD Display)**
1.  SAFE STATE:
                “All SAFE”

2. WARNING :
                “WARNING”
          Shows direction + distance
 
3. HIGH RISK :
                “HIGH RISK”
          Shows direction + distance

4. CRITICAL :
                “CRITICAL”
          Shows direction + distance

**Components**
1. Arduino UNO R3 (with USB cable)
2. HV-SR04 Ultrasonic Sensors (3 sensors)
3. 16x2 LCD Display (12C Module) 
4. Buzzer, Resistor (220 ohm) and Red LED
5. Wires and Breadboard 
6. Toy Bicycle Model 
7. 18650 3.7v Lithium Battery with holder and switch

**Installation**
1. Assemble the hardware components.
2. Open the Arduino IDE.
3. Upload the source code to the Arduino Uno.
4. Connect power and test the system.

**Performance Analysis**
1. Real-time response achieved 
2. Accurate distance detection 
3. Speed-based risk improves reliability 
4. Reduced false alerts using averaging
